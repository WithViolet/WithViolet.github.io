中国剩余定理

过程有些坎坷，因为忽视了exgcd里面一点小变化，半天无法通过样例
不过终于搞懂了

设硬币数为N，可得
N = k1x1 + y1
N = k2x2 + y2
…
N = knxn + yn

主要思路：

把多个式子化为一个，最后求N = km + r的N的最小解
那么首先，怎么把两个式子化为一个呢？
有 k1x1 + y1 = k2x2 + y2
k1x1 - k2x2 = y2 - y1
是不是可以套用exgcd求解呀
ax + by = n
a相当于x1
b相当于x2
x = k1, y = k2, n = y2 - y1
(为何不是y = -k2？k2的值可以为负数或正数，带不带负号无所谓)
设d = exgcd(m1,m2,k1,k2)
if(y2 - y1) % d != 0 return -1无解
如果有解，我们有公式，通解为
x = x0 + b / gcd t
y = y0 - a / gcd t(t为正整数)
我们要求出前面两个式子的关于N的最小解，再继续和后面的式子合并才能得到最小解对吧，那么我们要取x的最小值，也就是k1的最小值
我们根据题意有
k1 m1 + a1 = k2 m2 + a2
则k1m1 = a2 - a1(mod m2)
等式两边同时除以gcd
k1m1 = a2 - a1(mod m2 / gcd)

又比如：
N = k1m1 + r1
N = k2m2 + r2
把N看作解系
1.r1 = r2 = 0那么合并之后N = klcm(m1,m2) + 0
2.r1 != 0,r2 = 0,合并后 N = klcm(m1,m2) + r1
3.r1 = 0,r2 != 0,合并后 N = klcm(m1,m2) + r2
4.r1,r2 != 0,合并后 N = klcm(m1,m2) + gcd(m1,m2)

所以用k1m1 = a2 - a1(mod m2 / gcd)
求出k1之后，我们可以求出R
R = k1m1 + r1
N = kM + R
N = klcm(m1,m2) + R
lcm(m1,m2) = m1*m2/gcd
这样就可以求出合并后的式子，就可以继续和剩下的式子合并
